# WASP
WASP - Windows Aids for Stochastic Processes

An animated and visual version of binomial probability.

There is a 50/50 chance he'll grab a chocolate bar, or a cup of coffee. There's also a 20% chance his pants will fall down.

This is one part of a package that demonstrates Binomials, 2D walks, Markov Chains and Poisson distributions.
